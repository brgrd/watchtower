Below is a **copy‑paste‑ready implementation** of **Watchtower** — the agentic, RSS‑bounded infra‑sec monitor implemented **entirely on GitHub** (no Cloudflare). It includes the repo scaffold, complete workflow, runnable Python agent, config/state files, and a README. You can drop this into a new repo and feed it straight to Codex/Copilot.

> **Platform notes (for your reviewers):**
>
> *   GitHub Actions **cron schedules** run in **UTC** and **only trigger when the workflow file is on the repository’s default branch**. We therefore schedule hourly in UTC and gate to America/New\_York hours inside the job. [\[github.com\]](https://github.com/dhannusch/groq-docs-mcp), [\[console.groq.com\]](https://console.groq.com/docs/examples)
> *   We use workflow‑level **concurrency** to prevent overlapping runs. [\[bing.com\]](https://bing.com/search?q=Groq+Python+SDK+documentation)
> *   **GitHub Pages** deployment uses the official `actions/upload-pages-artifact` + `actions/deploy-pages@v4` and requires `pages: write` and `id-token: write`. [\[developers...dflare.com\]](https://developers.cloudflare.com/workers/examples/cron-trigger/)
> *   **Caching** uses `actions/cache@v5` for speed; the repo files remain the source of truth. [\[boristane.com\]](https://boristane.com/blog/what-are-cloudflare-durable-objects/)
> *   **Secrets & Variables:** Put `GROQ_API_KEY` in **repository secrets**, non‑secret config in **repository variables**. [\[developers...dflare.com\]](https://developers.cloudflare.com/d1/platform/pricing/)

## 2) Workflow — **`.github/workflows/agent.yml`**

```yaml
name: Watchtower — Agentic InfraSec Monitor

on:
  schedule:
    - cron: "5 * * * *"   # UTC; avoid top-of-hour congestion
  workflow_dispatch: {}

# Prevent overlapping runs
concurrency:
  group: watchtower-infrasec
  cancel-in-progress: false

# Least-privilege; Pages deploy needs pages+id-token
permissions:
  contents: write
  issues: write            # optional: if you later post summaries as issues
  pages: write             # required by deploy-pages
  id-token: write          # required by deploy-pages

jobs:
  run:
    runs-on: ubuntu-latest
    timeout-minutes: 15

    steps:
      - uses: actions/checkout@v4

      - name: Restore ephemeral cache
        uses: actions/cache@v5
        with:
          path: .cache/
          key: cache-${{ runner.os }}-${{ hashFiles('agent/**', 'requirements.txt') }}
          restore-keys: |
            cache-${{ runner.os }}-

      - name: Setup Python 3.11
        uses: actions/setup-python@v5
        with:
          python-version: "3.11"

      - name: Install deps
        run: |
          python -m pip install --upgrade pip
          pip install -r requirements.txt

      - name: Gate by America/New_York hours (0,6,12,18)
        id: gate
        run: |
          python - <<'PY'
import os, datetime, zoneinfo
ny_hour = datetime.datetime.now(zoneinfo.ZoneInfo("America/New_York")).hour
within = str(ny_hour in (0,6,12,18)).lower()  # 'true' or 'false'
with open(os.environ['GITHUB_OUTPUT'], 'a') as fh:
    fh.write(f"within={within}\n")
print(f"NY_HOUR={ny_hour} within={within}")
PY

      - name: Exit outside window
        if: steps.gate.outputs.within != 'true'
        run: echo "Skipping: outside approved NY hours"

      - name: Run Watchtower (plan → act → observe)
        if: steps.gate.outputs.within == 'true'
        env:
          GROQ_API_KEY: ${{ secrets.GROQ_API_KEY }}
        run: |
          python agent/runner.py

      - name: Commit report/state updates
        if: ${{ !cancelled() }}
        run: |
          if [ -n "$(git status --porcelain)" ]; then
            git config user.name  "github-actions[bot]"
            git config user.email "github-actions[bot]@users.noreply.github.com"
            git add reports/ state/ || true
            git commit -m "chore(watchtower): briefing update [skip ci]" || true
            git push
          fi

  deploy-pages:
    needs: run
    if: ${{ !cancelled() }}
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Upload briefing site
        uses: actions/upload-pages-artifact@v3
        with:
          path: reports
      - name: Deploy to GitHub Pages
        id: deployment
        uses: actions/deploy-pages@v4
```

> **Why this shape:** cron is UTC + requires default branch; we gate local hours at runtime. Concurrency prevents overlaps. Pages deploy uses official actions and requires the two permissions shown. Cache accelerates repeat runs. [\[github.com\]](https://github.com/dhannusch/groq-docs-mcp), [\[console.groq.com\]](https://console.groq.com/docs/examples), [\[bing.com\]](https://bing.com/search?q=Groq+Python+SDK+documentation), [\[developers...dflare.com\]](https://developers.cloudflare.com/workers/examples/cron-trigger/), [\[boristane.com\]](https://boristane.com/blog/what-are-cloudflare-durable-objects/)

## 5) Agent runtime — **`agent/runner.py`**

```python
#!/usr/bin/env python3
# Watchtower agent: plan → act → observe → (optional) re-plan
import os, sys, json, time, hashlib, re, textwrap
from datetime import datetime, timezone, timedelta
import yaml, requests, feedparser, tldextract
from bs4 import BeautifulSoup

ROOT = os.path.dirname(os.path.dirname(__file__))
REPORTS_DIR = os.path.join(ROOT, "reports")
STATE_DIR = os.path.join(ROOT, "state")
IGNORE_FILE = os.path.join(STATE_DIR, "ignore_registry.json")
LEDGER_FILE = os.path.join(STATE_DIR, "ledger.jsonl")
CONFIG = yaml.safe_load(open(os.path.join(ROOT, "agent", "config.yaml"), "r", encoding="utf-8"))

GROQ_API_KEY = os.getenv("GROQ_API_KEY")
GROQ_BASE = "https://api.groq.com/openai/v1"  # OpenAI-compatible Chat API

# -----------------------------
# Utilities
# -----------------------------
def now_utc_iso():
    return datetime.utcnow().replace(tzinfo=timezone.utc).isoformat()

def sha256(s: str) -> str:
    return hashlib.sha256(s.encode("utf-8", errors="ignore")).hexdigest()

def load_json(path, default):
    if not os.path.exists(path): return default
    with open(path, "r", encoding="utf-8") as f:
        try: return json.load(f)
        except: return default

def save_json(path, obj):
    with open(path, "w", encoding="utf-8") as f:
        json.dump(obj, f, indent=2, ensure_ascii=False)

def append_jsonl(path, obj):
    with open(path, "a", encoding="utf-8") as f:
        f.write(json.dumps(obj, ensure_ascii=False) + "\n")

# -----------------------------
# Safety & Validation
# -----------------------------
PRIVATE_NETS = [
    ("10.",), ("192.168.",), ("172.",), ("127.",), ("169.254.",), ("::1",),
]
EXECUTABLE_CT = (
    "application/x-msdownload","application/x-msdos-program","application/octet-stream",
    "application/x-7z-compressed","application/zip","application/x-tar","application/gzip"
)

def is_private_host(url: str) -> bool:
    host = requests.utils.urlparse(url).hostname or ""
    if host.endswith(".local") or host.endswith(".lan"): return True
    for p in PRIVATE_NETS:
        if host.startswith(p[0]): return True
    return False

def fetch_url(url: str, timeout=10, max_redirects=5):
    if not url.lower().startswith("https://"):
        raise ValueError("Non-HTTPS blocked by policy")
    if is_private_host(url):
        raise ValueError("Private network host blocked")
    session = requests.Session()
    session.max_redirects = max_redirects
    r = session.get(url, timeout=timeout, allow_redirects=True,
                    headers={"User-Agent":"Watchtower/1.0"})
    ct = (r.headers.get("content-type") or "").split(";")[0].strip().lower()
    if any(ct.startswith(bad) for bad in EXECUTABLE_CT):
        raise ValueError(f"Executable content-type blocked: {ct}")
    html = r.text
    if len(html.encode("utf-8")) > 2_000_000:
        raise ValueError("Document too large")
    soup = BeautifulSoup(html, "html.parser")
    text = " ".join(soup.get_text(" ").split())
    if len(text) < 800:
        raise ValueError("Insufficient text extracted")
    return text[:200_000]

def add_ignore(ignore, typ, key, ttl_days):
    bucket = f"ignore_{typ}"
    ttl = (datetime.utcnow() + timedelta(days=ttl_days)).date().isoformat()
    ignore.setdefault(bucket, {})
    ignore[bucket][key] = ttl

def is_ignored(ignore, url: str) -> bool:
    host = requests.utils.urlparse(url).hostname or ""
    prefix_hits = any(url.startswith(p) for p in ignore.get("ignore_url_prefix", {}))
    domain_hits = host in ignore.get("ignore_domain", {})
    url_hits = url in ignore.get("ignore_url", {})
    return prefix_hits or domain_hits or url_hits

# -----------------------------
# Feeds (RSS/JSON) and polling
# -----------------------------
def _poll_rss(url: str, since_hours: int, ignore: dict) -> list:
    """Parse an RSS/Atom feed via feedparser."""
    fp = feedparser.parse(url)
    items = []
    cutoff = datetime.utcnow() - timedelta(hours=since_hours)
    for e in fp.entries:
        link = getattr(e, "link", None) or getattr(e, "id", None)
        if not link: continue
        if is_ignored(ignore, link): continue
        published = None
        for k in ("published_parsed", "updated_parsed"):
            if getattr(e, k, None):
                published = datetime(*getattr(e, k)[:6])
                break
        if published and published < cutoff: continue
        items.append({
            "title": getattr(e, "title", "")[:200],
            "url": link,
            "summary": getattr(e, "summary", "")[:500],
            "source": url
        })
    return items

def _poll_nvd_api(url: str, since_hours: int, ignore: dict) -> list:
    """Poll NVD API 2.0 (https://services.nvd.nist.gov/rest/json/cves/2.0).
    Filters by pubStartDate/pubEndDate; paginates automatically.
    NVD enforces a 6-req/30 s rolling rate limit without an API key;
    set NVD_API_KEY env var to raise the limit to 50-req/30 s.
    """
    cutoff = datetime.utcnow() - timedelta(hours=since_hours)
    params = {
        "pubStartDate": cutoff.strftime("%Y-%m-%dT%H:%M:%S.000"),
        "pubEndDate": datetime.utcnow().strftime("%Y-%m-%dT%H:%M:%S.000"),
        "resultsPerPage": 100,
        "startIndex": 0,
    }
    api_key = os.getenv("NVD_API_KEY")  # optional; higher rate limit
    headers = {"User-Agent": "Watchtower/1.0"}
    if api_key:
        headers["apiKey"] = api_key
    items = []
    while True:
        r = requests.get(url, params=params, headers=headers, timeout=30)
        if r.status_code == 429:
            time.sleep(30)
            continue
        r.raise_for_status()
        data = r.json()
        for vuln in data.get("vulnerabilities", []):
            cve = vuln.get("cve", {})
            cve_id = cve.get("id", "")
            detail_url = f"https://nvd.nist.gov/vuln/detail/{cve_id}"
            if is_ignored(ignore, detail_url): continue
            desc = next(
                (d["value"] for d in cve.get("descriptions", []) if d.get("lang") == "en"),
                ""
            )
            items.append({
                "title": cve_id,
                "url": detail_url,
                "summary": desc[:500],
                "source": url
            })
        total = data.get("totalResults", 0)
        params["startIndex"] += data.get("resultsPerPage", 100)
        if params["startIndex"] >= total:
            break
        time.sleep(6)  # NVD rate-limit courtesy delay
    return items

def _poll_cisa_kev(url: str, ignore: dict) -> list:
    """Poll the CISA Known Exploited Vulnerabilities JSON catalog.
    The catalog has no per-entry date field; all entries are returned and
    deduplication is handled by the seen-hashes registry in state/.
    """
    r = requests.get(url, headers={"User-Agent": "Watchtower/1.0"}, timeout=30)
    r.raise_for_status()
    data = r.json()
    items = []
    for v in data.get("vulnerabilities", []):
        cve_id = v.get("cveID", "")
        detail_url = f"https://nvd.nist.gov/vuln/detail/{cve_id}"
        if is_ignored(ignore, detail_url): continue
        items.append({
            "title": f"{cve_id} — {v.get('vulnerabilityName', '')[:120]}",
            "url": detail_url,
            "summary": v.get("shortDescription", "")[:500],
            "source": url
        })
    return items

def poll_feed(feed_cfg: dict, since_hours: int, ignore: dict) -> list:
    """Dispatch to the correct poller based on feed type field."""
    url = feed_cfg["url"]
    feed_type = feed_cfg.get("type", "rss")
    try:
        if feed_type == "json_api":
            if "nvd.nist.gov" in url or "services.nvd.nist.gov" in url:
                return _poll_nvd_api(url, since_hours, ignore)
            elif "cisa.gov" in url and "known_exploited" in url:
                return _poll_cisa_kev(url, ignore)
            else:
                # Generic JSON API: best-effort parse of top-level list
                r = requests.get(url, headers={"User-Agent": "Watchtower/1.0"}, timeout=30)
                r.raise_for_status()
                raw = r.json()
                entries = raw if isinstance(raw, list) else raw.get("items", raw.get("entries", []))
                return [
                    {"title": str(e.get("title", ""))[:200],
                     "url": str(e.get("url", e.get("link", ""))),
                     "summary": str(e.get("summary", e.get("description", "")))[:500],
                     "source": url}
                    for e in entries if e.get("url") or e.get("link")
                ]
        else:
            return _poll_rss(url, since_hours, ignore)
    except Exception as exc:
        print(f"[WARN] poll_feed failed for {url}: {exc}")
        return []

# -----------------------------
# Planner (Groq)
# -----------------------------
def groq_chat(messages, model, temperature=0.2, max_tokens=1200):
    if not GROQ_API_KEY:
        raise RuntimeError("GROQ_API_KEY environment variable is not set")
    headers = {"Authorization": f"Bearer {GROQ_API_KEY}", "Content-Type": "application/json"}
    payload = {"model": model, "messages": messages, "temperature": temperature,
               "max_tokens": max_tokens, "stream": False}
    for attempt in range(5):
        res = requests.post(f"{GROQ_BASE}/chat/completions", headers=headers, json=payload, timeout=60)
        if res.status_code == 429:
            retry = int(res.headers.get("retry-after") or "2")
            time.sleep(min(30, retry + attempt))  # capped backoff
            continue
        res.raise_for_status()
        data = res.json()
        content = data["choices"][0]["message"]["content"]
        return content, {
            "rpd_rem": res.headers.get("x-ratelimit-remaining-requests"),
            "tpm_rem": res.headers.get("x-ratelimit-remaining-tokens")
        }
    raise RuntimeError("Groq rate limited repeatedly")
# Groq uses an OpenAI-compatible Chat Completions API and exposes rate-limit headers; backoff on 429. [7](https://graphite.com/guides/github-actions-caching)[8](https://bing.com/search?q=GitHub+Actions+concurrency+cancel-in-progress+docs)

def request_action_plan(goal, budget, context):
    sys_prompt = "You are a planner. Output only strict JSON per schema; no prose."
    user = {
        "run_goal": goal,
        "budget": budget,
        "context": {"recent_sources": [s["url"] for s in context[:10]]},
        "schema": {
            "steps": [
                {"tool":"POLL_FEED","args":{"feed_id":"int","since_hours":"int"}},
                {"tool":"ADD_FEED","args":{"url":"string","category":"string"}},
                {"tool":"CLUSTER","args":{"window_hours":"int"}},
                {"tool":"SELECT_SOURCES","args":{"cluster_id":"int"}}
            ]
        }
    }
    content, _ = groq_chat(
        [
            {"role":"system","content":sys_prompt},
            {"role":"user","content":json.dumps(user, ensure_ascii=False)}
        ],
        model=CONFIG["model"]["name"],
        temperature=0.1,
        max_tokens=600
    )
    try:
        plan = json.loads(content)
        assert "steps" in plan and isinstance(plan["steps"], list)
        return plan
    except Exception:
        return {
            "run_goal": goal,
            "budget": budget,
            "steps": [{"tool":"CLUSTER","args":{"window_hours":6}}]
        }

# -----------------------------
# Domain taxonomy helpers
# -----------------------------
_TAXONOMY = CONFIG.get("domain_taxonomy", {})

def classify_domains(item: dict) -> list:
    """Return a list of domain bucket keys that match this item's text.
    Multi-label: one item may belong to several buckets.
    Falls back to ['uncategorised'] if nothing matches.
    """
    txt = (item.get("title", "") + " " + item.get("summary", "")).lower()
    matched = [
        bucket
        for bucket, cfg in _TAXONOMY.items()
        if any(sig.lower() in txt for sig in cfg.get("signals", []))
    ]
    return matched or ["uncategorised"]

def build_domain_heatmap(cards: list) -> dict:
    """Aggregate cluster cards by domain bucket.
    Returns a dict keyed by bucket id:
      { bucket_id: {"label": str, "icon": str, "max_score": int, "count": int} }
    All configured buckets are always present (zero if no findings).
    """
    heatmap = {
        bk: {"label": cfg["label"], "icon": cfg.get("icon", ""),
              "max_score": 0, "count": 0}
        for bk, cfg in _TAXONOMY.items()
    }
    # Add a catch-all bucket for uncategorised findings
    heatmap["uncategorised"] = {"label": "Other", "icon": "❓",
                                  "max_score": 0, "count": 0}
    for card in cards:
        for bucket in card.get("domains", ["uncategorised"]):
            if bucket not in heatmap:
                continue
            heatmap[bucket]["count"] += 1
            heatmap[bucket]["max_score"] = max(
                heatmap[bucket]["max_score"], card["risk_score"]
            )
    return heatmap

# Clustering & Risk Scoring

def normalize_item_text(item):
    return " ".join([item.get("title", ""), item.get("summary", "")])

def cluster_items(items, window_hours=6):
    clusters = {}
    cve_re = re.compile(r"\bCVE-\d{4}-\d{4,7}\b", re.I)
    for it in items:
        txt = normalize_item_text(it)
        m = cve_re.search(txt)
        if m:
            key = f"CLUSTER:CVE:{m.group(0).upper()}"
        else:
            dom = tldextract.extract(it["url"]).registered_domain or "unknown"
            key = f"CLUSTER:{dom}:{sha256(it.get('title', ''))[:10]}"
        clusters.setdefault(key, []).append(it)
    return clusters

def score_cluster(key, items):
    score = 0
    if key.startswith("CLUSTER:CVE:"): score += 40
    text = " ".join(normalize_item_text(i) for i in items).lower()
    if "exploit" in text or "in the wild" in text: score += 30
    if "poc" in text: score += 15
    if any(w in text for w in ("supply chain", "dependency", "package")): score += 20
    if any(w in text for w in ("kubernetes", "istio", "nginx", "openssl", "linux kernel")): score += 15
    return max(0, min(100, score))

def to_cluster_card(key, items):
    score = score_cluster(key, items)
    title = items[0]["title"][:140] if items else key
    primary = [{"title": it["title"][:120], "url": it["url"]} for it in items[:5]]
    # Classify all items in this cluster and union their domain tags
    all_domains: list = []
    for it in items:
        for d in classify_domains(it):
            if d not in all_domains:
                all_domains.append(d)
    return {
        "id": sha256(key)[:12],
        "risk_score": score,
        "domains": all_domains,
        "badges": [],
        "title": title,
        "summary": f"{len(items)} related updates.",
        "what_new": [],
        "affected": {"ecosystems": [], "components": []},
        "sources": {"primary": primary, "secondary": []}
    }

SEEN_FILE = os.path.join(STATE_DIR, "seen_hashes.json")

def load_seen() -> set:
    d = load_json(SEEN_FILE, {"hashes": []})
    return set(d.get("hashes", []))

def save_seen(seen: set):
    # Bound file growth to 50 000 most-recent hashes
    recent = list(seen)[-50_000:]
    save_json(SEEN_FILE, {"hashes": recent})

def item_hash(item: dict) -> str:
    return sha256(item.get("url", "") + item.get("title", ""))

def deduplicate(items: list, seen: set) -> tuple:
    fresh = []
    for it in items:
        h = item_hash(it)
        if h not in seen:
            seen.add(h)
            fresh.append(it)
    return fresh, seen

def main():
    try:
        _run()
    except Exception as exc:
        # Always write a minimal ledger entry so failures are traceable
        os.makedirs(STATE_DIR, exist_ok=True)
        append_jsonl(LEDGER_FILE, {"ts": now_utc_iso(), "error": str(exc)})
        print(f"[ERROR] Watchtower run failed: {exc}", file=sys.stderr)
        sys.exit(1)

def _run():
    from concurrent.futures import ThreadPoolExecutor, as_completed
    from agent.planning import dispatch_plan

    os.makedirs(REPORTS_DIR, exist_ok=True)
    os.makedirs(STATE_DIR, exist_ok=True)
    ignore = load_json(IGNORE_FILE, {"ignore_url": {}, "ignore_domain": {}, "ignore_url_prefix": {}})
    seen = load_seen()

    budgets = CONFIG["budgets"]
    feeds_cfg = [f for f in CONFIG["feeds"] if f.get("enabled", True)]
    since_hours = budgets.get("since_hours", 6)   # config-driven; default 6 h
    run_deadline = time.monotonic() + budgets["max_runtime_seconds"]

    # 1) Poll feeds (bounded by max_feeds_polled + runtime budget)
    polled = []
    for fcfg in feeds_cfg[:budgets["max_feeds_polled"]]:
        if time.monotonic() > run_deadline:
            print("[WARN] Runtime budget reached during feed polling.")
            break
        polled += poll_feed(fcfg, since_hours, ignore)

    # 2) Deduplicate against seen-hashes registry
    polled, seen = deduplicate(polled, seen)
    save_seen(seen)

    # 3) Agent planning
    plan = request_action_plan(
        goal="Produce infrastructure security briefing",
        budget={"max_steps": budgets["max_agent_steps"],
                "max_url_fetches": budgets["max_url_fetches"],
                "max_new_feeds": budgets["max_new_feeds"]},
        context=polled
    )

    # 4) Dispatch plan steps (POLL_FEED, ADD_FEED; CLUSTER/SELECT_SOURCES deferred)
    polled = dispatch_plan(plan, polled, ignore, budgets, since_hours, run_deadline)

    # 5) Concurrent enrichment — bounded by workers + runtime budget
    enriched = []
    fetch_candidates = [it for it in polled if not is_ignored(ignore, it["url"])]
    fetch_candidates = fetch_candidates[:budgets["max_url_fetches"]]

    def _enrich(it):
        if time.monotonic() > run_deadline:
            return None, None, None
        url = it["url"]
        try:
            _ = fetch_url(url)
            it["extracted_text_hash"] = sha256((it["title"] + it.get("summary", ""))[:5000])
            return it, None, None
        except Exception as ex:
            host = requests.utils.urlparse(url).hostname or ""
            url_ign = ("url", url) if "Non-HTTPS" in str(ex) else None
            dom_ign = ("domain", host) if "Executable content-type" in str(ex) else None
            return None, url_ign, dom_ign

    with ThreadPoolExecutor(max_workers=budgets.get("max_fetch_workers", 6)) as pool:
        futures = {pool.submit(_enrich, it): it for it in fetch_candidates}
        for fut in as_completed(futures):
            item, url_ign, dom_ign = fut.result()
            if item: enriched.append(item)
            if url_ign: add_ignore(ignore, url_ign[0], url_ign[1], 30)
            if dom_ign: add_ignore(ignore, dom_ign[0], dom_ign[1], 90)

    save_json(IGNORE_FILE, ignore)

    # 6) Cluster & score
    clusters = cluster_items(enriched or polled, window_hours=since_hours)
    cards = [to_cluster_card(k, v) for k, v in clusters.items()]
    cards = sorted(cards, key=lambda c: c["risk_score"], reverse=True)[:budgets["max_clusters_output"]]

    # 7) Outputs
    ts = datetime.utcnow().strftime("%Y-%m-%d_%H-%M")
    latest_md  = os.path.join(REPORTS_DIR, "latest.md")
    out_md     = os.path.join(REPORTS_DIR, f"briefing_{ts}.md")
    out_jsonl  = os.path.join(REPORTS_DIR, f"briefing_{ts}.jsonl")
    index_html = os.path.join(REPORTS_DIR, "index.html")

    md_lines = ["---",
                f"generated_at: {now_utc_iso()}",
                f"model: {CONFIG['model']['name']}",
                "project: Watchtower",
                "---",
                "# Watchtower — Infrastructure Security Briefing",
                ""]
    for c in cards:
        md_lines.append(f"## {c['title']}  (risk: {c['risk_score']})")
        md_lines.append(c["summary"])
        md_lines.append("")
        for s in c["sources"]["primary"][:CONFIG["report"]["links_per_cluster_limit"]]:
            md_lines.append(f"- [{s['title']}]({s['url']})")
        md_lines.append("")

    md_content = "\n".join(md_lines)
    with open(out_md, "w", encoding="utf-8") as f: f.write(md_content)
    with open(latest_md, "w", encoding="utf-8") as f: f.write(md_content)
    with open(out_jsonl, "w", encoding="utf-8") as f:
        for c in cards: f.write(json.dumps(c, ensure_ascii=False) + "\n")

    # 8) Build domain heat map and regenerate index.html
    heatmap = build_domain_heatmap(cards)
    _write_index_html(index_html, cards, heatmap, ts)

    # 9) Append run record to ledger
    hot_domains = [k for k, v in heatmap.items() if v["max_score"] >= 60]
    append_jsonl(LEDGER_FILE, {
        "ts": now_utc_iso(),
        "counts": {"feeds": len(feeds_cfg), "polled": len(polled),
                   "enriched": len(enriched), "clusters": len(cards)},
        "hot_domains": hot_domains,
        "plan_steps": len(plan.get("steps", []))
    })

    # 10) Job summary
    print("### Watchtower run")
    print(f"UTC: {now_utc_iso()}")
    print(f"Feeds: {len(feeds_cfg)}  Fresh: {len(polled)}  Enriched: {len(enriched)}  Clusters: {len(cards)}")
    print("\nTop:")
    for c in cards[:5]:
        print(f"  - {c['title']} (risk {c['risk_score']})")


def _heatmap_cell_color(max_score: int, count: int) -> tuple:
    """Return (background_hex, text_hex) for a heat map cell.
    Scale: no findings → grey; low → green; medium → amber; high → orange; critical → red.

    if count == 0:    return "#ebedf0", "#57606a"   # grey  — no findings
```

```python
"""Toolbelt: concrete implementations of plan-step tools dispatched by planning.py."""
import re
from urllib.parse import urlparse
import feedparser, requests

ALLOWED_CATEGORIES = {"vulns", "advisories", "blog", "supply_chain", "cloud", "osint"}
URL_RE = re.compile(r'^https://[\w./\-?=&%+#:@]+$')


def tool_add_feed(url: str, category: str, feeds_cfg: list, max_new: int) -> tuple:
    """ADD_FEED: validate and append a new RSS feed to the in-memory feed list.
    Guards: HTTPS only, valid category, not already present, parseable as RSS/Atom,
    new-feed budget not exceeded.
    Returns (success: bool, reason: str).
    """
    if not URL_RE.match(url):
        return False, f"Invalid URL format: {url[:80]}"
    if urlparse(url).scheme != "https":
        return False, "Non-HTTPS rejected"
    if category not in ALLOWED_CATEGORIES:
        return False, f"Unknown category '{category}'; allowed: {sorted(ALLOWED_CATEGORIES)}"
    if any(f["url"] == url for f in feeds_cfg):
        return False, "Feed already registered"
    new_count = sum(1 for f in feeds_cfg if f.get("_dynamic"))
    if new_count >= max_new:
        return False, f"max_new_feeds budget ({max_new}) exhausted"
    try:
        fp = feedparser.parse(url)
        if fp.bozo and not fp.entries:
            return False, "URL does not appear to be a valid RSS/Atom feed"
    except Exception as exc:
        return False, f"Feed parse error: {exc}"
    feeds_cfg.append({
        "id": f"dynamic_{len(feeds_cfg)}",
        "url": url,
        "category": category,
        "type": "rss",
        "enabled": True,
        "_dynamic": True
    })
    return True, "Feed added"


def tool_select_sources(cluster_id: str, cards: list, ignore: dict,
                        fetch_url_fn, add_ignore_fn) -> list:
    """SELECT_SOURCES: for a given cluster ID fetch secondary sources not yet in
    the primary list and attach them to the cluster card (up to 3 new links).
    Returns the updated cards list.
    """
    for card in cards:
        if card["id"] == cluster_id:
            primary_urls = {s["url"] for s in card["sources"]["primary"]}
            secondary = []
            for s in card["sources"].get("secondary_candidates", []):
                url = s.get("url", "")
                if not url or url in primary_urls:
                    continue
                try:
                    fetch_url_fn(url)
                    secondary.append({"title": s.get("title", url[:80]), "url": url})
                    if len(secondary) >= 3:
                        break
                except Exception as exc:
                    if "Executable content-type" in str(exc):
                        host = urlparse(url).hostname or ""
                        add_ignore_fn(ignore, "domain", host, 90)
            card["sources"]["secondary"] = secondary
            break
    return cards
```

**`agent/planning.py`**

```python
"""Planning helpers: dispatch_plan executes steps returned by request_action_plan."""
import sys, time
from agent.toolbelt import tool_add_feed, tool_select_sources


def dispatch_plan(plan: dict, polled: list, ignore: dict, budgets: dict,
                  since_hours: int, run_deadline: float) -> list:
    """Walk plan['steps'] and execute each recognised tool action.
    Unknown tools are skipped with a warning.
    CLUSTER and SELECT_SOURCES are deferred to the post-cluster phase in runner.py
    and are intentionally no-ops here.
    Returns the (possibly augmented) polled item list.
    """
    # Late import to avoid circular dependency; runner imports planning.
    from agent import runner

    steps = plan.get("steps", [])
    step_budget = budgets.get("max_agent_steps", 20)
    feeds_cfg = runner.CONFIG["feeds"]

    for i, step in enumerate(steps[:step_budget]):
        if time.monotonic() > run_deadline:
            print("[WARN] Runtime budget reached during plan dispatch.")
            break

        tool = step.get("tool", "")
        args = step.get("args", {})

        if tool == "POLL_FEED":
            feed_id = args.get("feed_id")
            sh = int(args.get("since_hours", since_hours))
            for fcfg in feeds_cfg:
                if fcfg.get("id") == feed_id and fcfg.get("enabled"):
                    extra = runner.poll_feed(fcfg, sh, ignore)
                    polled.extend(extra)
                    print(f"[POLL_FEED] {feed_id}: +{len(extra)} items")
                    break

        elif tool == "ADD_FEED":
            ok, reason = tool_add_feed(
                url=args.get("url", ""),
                category=args.get("category", "osint"),
                feeds_cfg=feeds_cfg,
                max_new=budgets.get("max_new_feeds", 3)
            )
            print(f"[ADD_FEED] {'OK' if ok else 'SKIP'}: {reason}")
            if ok:
                new_feed = feeds_cfg[-1]
                extra = runner.poll_feed(new_feed, since_hours, ignore)
                polled.extend(extra)

        elif tool in ("CLUSTER", "SELECT_SOURCES"):
            pass  # handled post-dispatch in runner._run()

        else:
            print(f"[WARN] Unknown plan tool '{tool}' at step {i} — skipped",
                  file=sys.stderr)

    return polled
```

    *   `reports/latest.md`, timestamped `briefing_*.md`, `briefing_*.jsonl`.
    *   `reports/index.html` — GitHub Pages landing page with a domain heat map grid at the top

# Watchtower — Infrastructure Security Briefing

## Analyst Summary

The current threat landscape is dominated by vulnerabilities in web frameworks and applications, with a high risk of exploitation in the wild. The most at-risk technology stacks include web applications and frameworks, particularly those using JavaScript runtimes. The single most urgent action item for a security team is to patch or mitigate the recently disclosed vulnerabilities in Hono and BioStar 2.

## CVE-2025-41257: BioStar 2 Vulnerability (risk: 85)
Suprema's BioStar 2 in version 2.9.11.6 allows users to set new passwords without providing the current one, allowing attackers to gain unauthorized access. A patch is available, but exploitation in the wild has been reported.

- [CVE-2025-41257](https://nvd.nist.gov/vuln/detail/CVE-2025-41257)

## CVE-2026-29086: Hono Web Application Framework Vulnerability (risk: 80)
Prior to version 4.12.4, Hono's setCookie utility did not validate cookie values, allowing attackers to inject malicious data. A patch is available, but exploitation in the wild has been reported.

- [CVE-2026-29086](https://nvd.nist.gov/vuln/detail/CVE-2026-29086)

## CVE-2026-29045: Hono Web Application Framework Vulnerability (risk: 75)
Prior to version 4.12.4, Hono's serveStatic feature is vulnerable to a bug that allows attackers to access sensitive information. A patch is available, but exploitation in the wild has been reported.

- [CVE-2026-29045](https://nvd.nist.gov/vuln/detail/CVE-2026-29045)

## CVE-2026-26002: Open OnDemand Files Application Vulnerability (risk: 70)
The Files application in Open OnDemand versions prior to 4.0.9 and 4.1.3 is susceptible to a vulnerability that allows attackers to access sensitive information. A patch is available, but exploitation in the wild has been reported.

- [CVE-2026-26002](https://nvd.nist.gov/vuln/detail/CVE-2026-26002)

## CVE-2026-2297: CPython Legacy File Loader Vulnerability (risk: 70)
The import hook in CPython that handles legacy *.pyc files is incorrectly handled, allowing attackers to access sensitive information. A patch is available, but exploitation in the wild has been reported.

- [CVE-2026-2297](https://nvd.nist.gov/vuln/detail/CVE-2026-2297)
